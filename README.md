# Income Expense Calculator

A simple Income Expense Calculator built using HTML, CSS, and JavaScript.  
This application helps users track income and expenses with full CRUD functionality.

# Features
- Add, edit, and delete income & expense entries
- Filter entries (All / Income / Expense)
- Automatic calculation of total income, expenses, and balance
- Data persistence using localStorage
- Responsive design for mobile and desktop

# Tech Stack
- HTML
- CSS
- JavaScript

# How to Use
1. Enter description and amount
2. Select income or expense
3. Click Add
4. Use filters to view data
5. Edit or delete entries anytime
